capitalize()

Inflector.php : 286

Uppercases words with configurable delimiters between words.

public capitalize(string $string[, string $delimiters = " -" ]) : string

Takes a string and capitalizes all of the words, like PHP's built-in ucwords function. This extends that behavior, however, by allowing the word delimiters to be configured, rather than only separating on whitespace.

Here is an example:

capitalize($string); // Top-O-The-Morning To All_of_you! echo $inflector->capitalize($string, '-_ '); // Top-O-The-Morning To All_Of_You! ?>
